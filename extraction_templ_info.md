# Data Extraction Preparation Script

Python script for preparing structured datasets for manual data extraction in systematic reviews.

This tool transforms screened bibliographic records into organized extraction tables, prioritizes studies, separates review articles, and supports scalable manual processing through batching.

---

## Purpose

The script supports the data extraction stage by:

- filtering relevant studies from screened records  
- separating review articles  
- prioritizing records for reading  
- generating structured extraction templates  
- dividing large datasets into manageable batches  

The output is designed to **support manual full-text data extraction**, not to replace it.

---

## Input Data

The script expects:

- `screened_records.xlsx`

This dataset should be generated by the screening script and must include:

- `TI` (Title)  
- `PY` (Year)  
- `DO` (DOI)  
- `screening_decision`  
- `relevance_score`  

Optional (auto-filled if missing):

- `platform`, `sensor_type`, `sensor_mode`  
- `application_domain`, `scaling`  
- `uav_applicability`, `must_cite`  
- `is_review`  

---

## Processing Workflow

The script performs the following steps:

1. **Data cleaning**
   - removes empty rows  
   - removes records without title  
   - ensures valid relevance scores  

2. **Review article separation**
   - records flagged as `is_review = True` are extracted  
   - excluded from further processing  
   - saved separately  

3. **Priority assignment**
   - HIGH → must-cite studies  
   - MEDIUM → relevance score ≥ 7  
   - LOW → remaining records  

4. **Dataset filtering**
   - **Full dataset** → include + maybe  
   - **Refined dataset** → include only  

5. **Sorting (refined dataset)**
   - by priority (HIGH → LOW)  
   - then by relevance score  

6. **Batch creation**
   - refined dataset split into groups (~50 records)  
   - supports stepwise manual review  

7. **Template generation**
   - structured extraction tables created  
   - key fields pre-filled  
   - manual fields left empty
